# 情感沟通模拟器

一个沉浸式情感沟通练习应用，通过高拟真的对话模拟提升用户的沟通能力。

## 项目概览

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS 4
- **UI组件**: shadcn/ui
- **LLM**: coze-coding-dev-sdk (豆包模型)

## 核心功能

1. **场景选择**: 事后挽回、冷战破冰、敏感话题
2. **伴侣类型**: 冷战型、爆发型、回避型
3. **性别选择**: 男性/女性，影响对话语气和语音音色
4. **对话模拟**: 模拟微信聊天界面
5. **情绪系统**: 后台隐藏数值计算情绪状态
6. **语音功能**: TTS 根据情绪状态和性别调整语速、音量、音色
7. **结局判定**: 和解成功 / 关系破裂

## 关键文件说明

### src/lib/types.ts
定义所有类型：SceneType、PartnerType、EmotionState、Message、GameSession 等

### src/lib/game-data.ts
游戏配置数据：场景配置、伴侣配置、情绪阈值、表情包映射、情绪计算逻辑

### src/app/api/chat/route.ts
后端API，处理对话请求：
- 接收用户消息
- 调用 LLM 生成回复
- 调用 TTS 生成语音
- 计算情绪变化
- 判断游戏状态

### src/app/api/tts/route.ts
后端API，处理语音合成请求：
- 根据情绪状态调整语速和音量
- 根据伴侣类型选择不同的声音

## 情绪系统

后台隐藏数值系统：
- **愤怒值 (anger)**: 0-100，越高越生气
- **信任值 (trust)**: 0-100，越高越信任
- **委屈值 (sadness)**: 0-100

情绪状态根据数值计算：
- `calm`: 平静
- `upset`: 委屈
- `angry`: 愤怒
- `sad`: 伤心
- `reconciled`: 和解 (通关成功)
- `broken`: 破裂 (通关失败)

## 语音功能 (TTS)

AI 回复会自动生成语音，根据情绪状态调整：
- **calm**: 正常语速、正常音量
- **upset**: 稍慢语速、稍低音量（表示委屈）
- **angry**: 快语速、较高音量（表示愤怒）
- **sad**: 慢语速、低音量（表示伤心）
- **reconciled**: 温和语速、正常音量
- **broken**: 很慢语速、很低音量（表示冷漠）

根据性别选择不同的声音：
- **男性**: 云舟男声
- **女性**: 迷人女友音色

前端支持：
- 自动播放语音（可关闭）
- 手动点击播放按钮重听

## 性别与语气

根据性别调整对话语气风格：
- **男性**: 直接、简洁、不撒娇，表达情绪时偏内敛或直接
- **女性**: 可以撒娇、表达细腻情感，情绪表达更丰富

## 注意事项

- LLM 和 TTS 调用必须在后端进行，不要在前端暴露 API
- 表情包使用微信风格表情符号
- 回复延迟根据情绪状态模拟真实感
- TTS 失败不影响主流程，会继续返回文本响应
